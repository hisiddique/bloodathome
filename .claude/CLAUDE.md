# BloodAtHome - Laravel 12 Phlebotomist Booking System

## Project Overview
A Laravel 12 application for booking phlebotomist services for at-home or clinic blood tests.

## Tech Stack
- Laravel 12
- Inertia.js (React)
- PHP 8.2+
- Laravel Fortify (authentication)

## Database Schema

### Core Tables
1. **phlebotomists** - Phlebotomist profiles with ratings, pricing, availability
2. **blood_tests** - Available blood test catalog with pricing
3. **labs** - Lab locations for clinic visits
4. **bookings** - Patient appointment bookings
5. **chat_messages** - Chat between patients and phlebotomists
